* Создал EC2 инстанс на AWS

* Обновил пакеты всей системы

* Установил пакет openjdk-17-jdk

* Установил Git

* Установил PostgreSQL, создал пользователя и БД

* Склонировал репозиторий

* Скачал мавен, собрал проект, запустил
